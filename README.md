# 🚛 Portfólio Profissional — Cleber Nascimento

Site de apresentação profissional desenvolvido para destacar a trajetória,
competências e resultados mensuráveis de Cleber Nascimento, especialista
sênior em Gestão de Operações Logísticas e Frotas com mais de 10 anos
de experiência na Ghisolfi Operações.

---

## 👤 Sobre o Profissional

Cleber Nascimento é especialista na metodologia Ambev (DPO), com histórico
comprovado de reversão de unidades críticas, padronização de operações e
redução de custos. Ao longo de uma década na Ghisolfi, ascendeu de
Supervisor/Coordenador a Gerente Regional de Frota, gerenciando 390
veículos distribuídos em 4 unidades no Nordeste.

---

## 📊 Principais Resultados Apresentados

| Indicador | Resultado |
|---|---|
| Economia total acumulada | R$ 150.000 |
| Redução no consumo de combustível | -6% |
| Tempo de liberação de veículos | 72min → 27min ✓ Meta superada |
| Índice de socorro em rota | 1,2% → 0,49% ✓ Meta superada |
| QLP (motoristas por caminhão) | 1,08 → 1,35 |
| Unidades padronizadas (DPO Sustentável) | 4 unidades / 390 veículos |

## 🛠️ Tecnologias Utilizadas

- **React** + **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Recharts** — gráficos interativos
- **jsPDF** — geração de currículo em PDF

## 🚀 Como Rodar Localmente

```bash
# Clone o repositório
git clone https://github.com/g1000py/portfolio-cleber-final.git

# Entre na pasta
cd portfolio-cleber-final

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
```

Acesse em: `http://localhost:5173`
